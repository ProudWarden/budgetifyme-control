# BudgetifyMe Control Center

## Purpose
Control center for managing BudgetifyMe product direction, development, and execution.

## Current Version
1.1.0 (In Progress)

## Product Vision
A simple, private expense tracking app with smart AI insights.

## Core Principles
- Simple and fast
- Privacy-first (no bank connection)
- Local storage is primary
- AI is assistive, not core

## Current Focus (v1.1.0)
- AI Insights (basic)
- Onboarding improvement
- Firebase (light integration)

## Banned Scope
- No AI chat
- No multi-language
- No redesign
- No new features outside defined scope

## Execution Rule
Ship fast. Keep it simple. No overengineering.
